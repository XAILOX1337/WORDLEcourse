Компьютерная игра «Wordle»: Суть игры заключается в угадывании слова, загаданного программой, за ограниченное количество попыток. При вводе слова программа предоставляет цветовую индикацию:
- Зеленый цвет — буква присутствует в слове и стоит на верной позиции.
- Желтый цвет — буква присутствует в слове, но находится в другой позиции.
- Серый цвет — буква отсутствует в загаданном слове.

1 версия: ✅
-	случайный выбор слова из встроенного словаря;
-	проверка введенного пользователем слова на совпадение с загаданным.;
-	визуальное отображение угаданных букв и их позиций с помощью цветовой индикации.

2 версия:
-	отображение клавиатуры с цветовой индикацией угаданных букв;✅
-	анимация раскрытия слова;✅
-	возможность ввода слова с виртуальной клавиатуры;✅
-	возможность повторной игры без перезапуска файла;✅
-	игра завершает свою работу в случае закрытия игрового окна вручную;✅
-	справка;
-	возможность выбора темы;
-	программа должна сохранять рекорд пользователя.
